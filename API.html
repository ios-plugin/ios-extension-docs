API

Class Method: import(moduleName)

加载一个子模块

  • moduleName : String, 要加载的子模块的name
  • 返回值: Boolean , 加载是否成功

示例

uex.import("HTTP")

Class Method: log(message)

输出系统日志,此日志内容可以在设备的实时日志或者Xcode控制台中查看

  • message : String, 日志内容

示例

uex.log("Hello World!")
